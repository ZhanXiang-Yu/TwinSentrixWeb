# failure risk
Failure risk is the model predicted likelihood that a machine will enter a failure or degraded state inside the prediction horizon.

# remaining useful life
Remaining useful life, or RUL, estimates how long a machine can continue before intervention may be required.

# downtime
Downtime is predicted lost operating time caused by a fault, jam, or required intervention.

# queue utilization
Queue utilization is queue length divided by queue capacity. High utilization indicates possible congestion or bottleneck behavior.

# throughput
Throughput is the predicted production rate in units per hour.

# baseline throughput
Baseline throughput is the expected healthy production rate used for comparison.

# model uncertainty
Model uncertainty indicates how confident the model is in the prediction. Higher values mean the prediction should be checked carefully.

# blocking
Blocking occurs when a station cannot discharge product because downstream equipment or queue space is unavailable.

# starvation
Starvation occurs when a station cannot process because upstream product is not available.

# failure mode
Failure mode is the predicted category of degradation, such as overheating, jam risk, sensor drift, or mechanical wear.
